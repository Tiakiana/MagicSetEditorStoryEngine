# Story Narrative — Front Style

## Fields

- **Title**: centered in the translucent banner.
- **Card Number**: number before the slash.
- **Card Total**: number after the slash.
- **Front Illustration**: full-card background image.

The numbering is rendered as:

`N / X`

## Install

Replace `story_narrative.mse-game` with the updated package and add
`story_narrative-front.mse-style` to MSE's `data` folder.

The existing Text and Image Narrative styles are not changed.
